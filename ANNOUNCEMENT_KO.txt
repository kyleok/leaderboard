DGM Spring 2026 - 얼굴 이미지 생성 챌린지
============================================================

개요

Deep Generative Models 수업의 일환으로 얼굴 이미지 생성 경진대회에 참가합니다.
생성 모델을 학습한 뒤 생성한 얼굴 이미지 1,000장을 리더보드에 제출하는 것이 목표입니다.
팀은 4가지 지표로 평가되며, 전체 지표의 평균 순위(Total)를 기준으로 랭킹이 결정됩니다.

리더보드: https://leaderboard.lait-lab.com

  종료일                 : 2026년 6월 10일

참가 방법

이 대회는 개인전으로, 각 학생이 단독 팀으로 참가합니다.

  1. https://leaderboard.lait-lab.com 접속
  2. 아래 정보를 입력합니다:
       - 초대 코드    : xUf8hxhX
       - 팀 이름      : 본인 이름을 Firstname_Lastname 형식으로 입력
                        (예: Hayeon_Kim, Jaejun_Yoo)
       - 표시 이름    : 팀 이름과 동일하게 입력해도 됩니다
       - 이메일 (선택): 입력 시 제출 완료 후 점수 결과를 이메일로 받을 수 있습니다
  3. "Join" 버튼 클릭 - 참가 완료!

  참고: 브라우저를 닫고 다시 접속할 경우, 동일한 초대 코드와 팀 이름을 입력하면
        세션이 복구됩니다.

  주의: 팀 이름은 한 번 등록하면 변경할 수 없습니다. 신중하게 입력해 주세요.

------------------------------------------------------------
제출 형식

  - 생성한 이미지를 담은 .zip 파일 1개
  - 이미지 형식: JPEG 또는 PNG
  - 최소 해상도: 64x64 (높을수록 유리 - 레퍼런스는 256x256)
  - 이미지 1,000장 권장 (최대 허용: 1,000장)
  - 최대 파일 크기: 200 MB
  - zip 내부에 하위 폴더를 만들지 말고 이미지를 루트에 직접 위치시켜 주세요

  zip 구조 예시:
    submission.zip
    |-- img_0000.jpg
    |-- img_0001.jpg
    |-- ...
    |-- img_0999.jpg

------------------------------------------------------------
평가 지표

제출 이미지는 CelebV-HQ 데이터셋의 실제 얼굴 이미지 32,550장과 비교됩니다.

  지표    | 방향        | 측정 내용
  --------|-------------|-----------------------------------------------
  FID     | 낮을수록 좋음 | 특징 분포 거리 (전반적 품질)
  IS      | 높을수록 좋음 | 이미지 품질 및 다양성 (Inception Score)
  KID     | 낮을수록 좋음 | 커널 기반 분포 거리 (비편향 FID 변형)
  TopPR   | 높을수록 좋음 | 충실도 + 다양성 균형 (Top-P & Top-R F1)
  Total   | 낮을수록 좋음 | 4개 지표 순위 평균 (최종 랭킹 기준)

Total 순위는 각 지표별 순위의 평균으로 계산됩니다.
평균 순위가 낮을수록 전체 성능이 좋습니다.

------------------------------------------------------------
팁

  - 리더보드 상단 드롭다운에서 지표별(FID / IS / KID / TopPR) 순위 확인 가능
  - 제출 후 약 1분 이내에 점수가 업데이트됩니다
  - 지표별 최고 점수만 리더보드에 표시됩니다
  - 이전 ZIP 파일은 자동으로 삭제되며 최신 제출 파일만 저장됩니다
  - 지표 점수가 -1로 표시되면 처리 오류입니다 (주로 이미지 수 부족)

자주 묻는 질문

Q: 혼자 참가할 수 있나요?
A: 네. 본인 이름으로 팀을 만들어 참가하면 됩니다.

Q: 팀을 변경할 수 있나요?
A: 아니요. 한 번 팀에 가입하면 변경이 불가능합니다. 신중하게 선택해 주세요.

Q: 제출이 실패하면 어떻게 하나요?
A: 리더보드에서 해당 제출 항목의 오류 메시지를 확인해 주세요.
   주요 원인: 잘못된 파일 형식, 이미지 2장 미만, 파일 크기 초과.

Q: 이미지 생성에 GPU가 필요한가요?
A: 모델 학습 시에는 필요합니다. 평가는 리더보드 서버에서 처리하므로
   ZIP 파일만 제출하면 됩니다.
